Pattern

Why

How

Character Represents
~ One alpha character (i.e., A-Z, a-z)
# One numeric character (i.e., 0-9)
* One alphanumeric character (i.e., A-z, a-z, 0-9)

Examples

Pattern Represents
##/##/## Date
0# ## ## ## ## French phone number
~~ #### Flight number
