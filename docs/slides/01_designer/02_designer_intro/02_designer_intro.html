Search

Create

  • Questionnaire title. Display name of questionnaire (e.g., Zimbabwe PICES 2023)
  • Questionnaire variable. File name that captures main level of data (e.g., households -> households.dta)

Share

  • Open questionnaire
  • Click on SETTINGS in upper right-hand corner

  1. Select access level for collaborator: Edit or View.
  2. Compose email or user name associated with Survey Solutions account.
  3. Click INVITE to grant the specified access and have Survey Solutions send an invitation email.
